## Features

- 🔐 Secure login with email/password + Remember Me
- 📸 Photos gallery with grid view and preview
- 🎵 Music player with playlist support and audio controls
- 🎬 Video uploads and playback
- 👤 Contacts via .vcf files only
- 📄 Documents: PDF, Word, Excel, TXT, APK, EXE (x86/x64)
- 🔗 Important links manager
- 📁 Custom folders in every section
- ↔️ File transfer between platform users (by email)
- 🔔 Real-time notification center with sound
- 📱 Fully mobile-responsive design
- 🌙 Beautiful animated dark UI

## How to Deploy on GitHub Pages

1. Upload all files to a GitHub repository
2. Go to **Settings → Pages**
3. Source: `main` branch, root `/`
4. Your site will be live at `https://yourusername.github.io/repo-name`

---

## Privacy

All data is stored **locally in the browser** via `localStorage`. No server required. Each user's data is isolated by their email address.
